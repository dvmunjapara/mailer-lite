### Requirements
* PHP 8.1
* Node ^16.0

<a name="step2"></a>
### Installation
* cp .env.example .env
* configure .env
* composer install
* php artisan migrate
* php artisan db:seed
* yarn

<a name="step4"></a>
### Test
* php artisan test
 
<a name="step4"></a>
### Run
* php artisan serve
* yarn build
